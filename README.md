# Premium Hardware eStore 🚀

A high-end, minimalist electronics component marketplace built with professional grade hardware enthusiasts in mind. This project is optimized for performance, security, and cloud scalability.

## 🛠️ Technology Stack
- **Engine**: PHP 8.5+ (optimized for Vercel Serverless Functions)
- **Database**: TiDB Cloud (MySQL compatible) with SSL/TLS enforcement
- **Hosting**: Vercel (Production grade build & routing)
- **Styling**: Vanilla CSS with a custom "Orange/Beige/Navy" premium palette

## 🚀 Key Features
- **Modern Hardware Catalog**: Curated selection of high-end PC components.
- **Secure Cloud Integration**: Seamless connection to distributed MySQL clusters.
- **Production-Ready Routing**: Clean URL structure using Vercel Serverless runtimes.
- **Minimalist UX**: Clean, distraction-free interface for serious builders.

## ⚙️ Deployment Settings
To deploy this project to your own Vercel environment, ensure the following environment variables are set:
- `DB_HOST`: Your cloud database host
- `DB_PORT`: Database port (default 4000 for TiDB)
- `DB_NAME`: Database name
- `DB_USER`: Database username
- `DB_PASS`: Database password
- `VERCEL`: `1`
