# Hardware eStore

A e-commerce storefront for high-end PC and electronics components, built in plain PHP and deployed as serverless functions on Vercel.

## Features

- Product catalog with categories, detail pages, and search

- User registration, login, and session-based authentication

- Cart-free direct checkout with order placement and status tracking

- Wishlist support per user

- Admin views for managing products and order status

- Responsive, dependency-free front end (vanilla CSS/JS)

## Tech Stack

| Layer      | Choice                                             |

|------------|-----------------------------------------------------|

| Language   | PHP 8.1+                                             |

| Database   | MySQL-compatible (tested against TiDB Cloud, SSL)    |

| Hosting    | Vercel (`vercel-php` serverless runtime)              |

| Front end  | Vanilla HTML/CSS/JS, no build step                    |
